    算法学习第四章总结

    分治策略 在分治策略中，我们在每层递归应用了如下三个步骤： 分解：将问题划分为一些子问题，形式一样，规模更小 解决：步骤递归的求解出子问题，如果子问题够小，则停止递归 合并：步骤将子问题的解组成原问题的解 4.1 最大子数组问题 问题：我们获得一个数组，需要得到其中最大的子数组 将问题分解，得到三个子问题： 最大子数组位于 left 到mid 之间 最大子数组在mid+1到right之间 最大子数组跨越了mid 在这三个子问题中取得最大值即可，分析三种情况，1，2都可以通过递归解决，第三种情况虽然不是基本问题（递归足够小时可以直接计算的情况），但是不可以通过递归来得到结果，但是，其加入了限制，即子数组必须跨越中点，我们可以对其设计一个$O(n)$的算法来计算跨越中点的子数组。 void find_max_crossing_subarray(int A[],int low,int mid,int high ,int &amp;max_left,int &amp;max_right,int &amp;sum_max){ int left_sum = -1000000; int sum = 0; max_left = 0; for(int i=mid;i&gt;low;i--){ sum = sum + A[i]; if(sum &gt; left_sum){ left_sum = sum; max_left = i; } } int right_sum = -1000000; for(int j=mid;j&lt;high;j++){ sum = sum+A[j]; if(sum &gt; right_sum){ right_sum = sum; max_right = j; } } sum_max = right_sum + left_sum; } 我们有了一个线性的计算跨越中点的最大子数组的算法，我们很容易的得到寻找整个子数组的算法。
