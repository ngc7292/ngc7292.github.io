有关python的一些问题汇总

装饰器

装饰器在开发中应用广泛，其多用于增强函数的行为以及增加复用。

在flask中，路由可以用装饰器来定义，可以减少开发者使用框架的工作量。

@app.route("/")
def index():
  ...

与装饰器有关的几个概念。

闭包： 闭包就是能够读取其他函数内部变量的函数。

eg：

def outer_func():
    loc_list = []
    def inner_func(name):
    	loc_list.append(len(loc_list) + 1)
			print '%s loc_list = %s' %(name, loc_list)
      return inner_func
    
clo_func_0 = outer_func()
clo_func_0('clo_func_0')
clo_func_0('clo_func_0')
clo_func_0('clo_func_0')
clo_func_1 = outer_func()
clo_func_1('clo_func_1')
clo_func_0('clo_func_0')
clo_func_1('clo_func_1')

上面的loc_list 就是闭包的自由变量，而且每个实例的自由变量不相关。

nonlocal ：python3中刚引入的声明，其作用是将变量标记为自由变量。简单来说，就是将不可变变量比如字符串元组等可以在闭包中更新。

装饰器就是动态的为对象添加额外的职责。

@decorater
def func():
  ...

就相当于

decorater(func)

协程

协程核心就是上下文切换，通过python中关键字yield可以进行流控制使得效率进一步提高。

在协程之前，多线程之间的交互是要通过锁来完成的。
