vue 学习笔记啊

Aug 13, 2019 00:00 · 58 words · 1 minute read vue

  • vue learn

vue learn

v-if, v-else-if, v-else 没有 有条件的渲染 <g v-if="flourish === 'A'"></g><g v-else-if="flourish === 'B'"></g><g v-else></g>
v-bind ： 动态绑定属性，或传递道具 <div :style="{ background: color }"></div>
v-on @ 将事件侦听器附加到元素 <button @click="fnName"></button>
v-model 没有 创建双向绑定 <textarea rows="5" v-model="message" maxlength="72"></textarea>
v-pre 没有 跳过编译原始内容，可以提高性能 <div v-pre>{{ raw content with no methods}}</div>
v-once 没有 不要退缩 <div class=”v-once”>Keep me from rerendering</div>
v-show 没有 将基于状态显示或隐藏组件/元素，但是将其保留在DOM中而不卸载（与v-if不同） <child v-show=”showComponent”></child>（当showComponent为true时切换可见性）
